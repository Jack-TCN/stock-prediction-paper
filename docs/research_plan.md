# 研究计划

## 1. 论文题目

`LSTM与GRU在股票价格预测中的对比研究：技术指标融合与Dropout正则化分析`

## 2. 研究目标

这篇论文不追求提出复杂的新模型，而是通过规范的对比实验回答一个更适合学生完成的问题：

在相同股票数据、相同训练流程和相同评价指标下，LSTM 与 GRU 哪一种更适合用于股票价格预测；技术指标融合和 Dropout 正则化是否能稳定提升预测表现。

## 3. 研究问题

### RQ1

在相同数据和相同实验设置下，`LSTM` 和 `GRU` 哪个模型对股票价格预测更准确？

### RQ2

在基础 `OHLCV` 特征上加入 `RSI`、`MACD` 和 `Bollinger Bands` 后，模型预测性能是否提升？

### RQ3

在模型中加入 `Dropout` 正则化后，是否能够降低过拟合并提升测试集表现？

### RQ4

`模型类型`、`特征方案` 和 `正则化策略` 的不同组合中，哪一种组合在不同股票上表现最稳定？

## 4. 研究设计

### 4.1 自变量

- 模型：`LSTM`、`GRU`
- 特征：`OHLCV`、`OHLCV + 技术指标`
- 正则化：无 Dropout、`Dropout=0.2`

### 4.2 因变量

- `MAE`
- `RMSE`
- `MAPE`
- `R²`

### 4.3 控制变量

- 股票：`AAPL`、`MSFT`、`TSLA`
- 时间范围：`2015-01-01` 到 `2025-12-31`
- 预测目标：下一交易日收盘价
- 窗口长度：`20`
- 切分方式：按时间顺序切分 `70% / 15% / 15%`
- 归一化：`MinMaxScaler`
- 优化器：`Adam`
- 损失函数：`MSE`
- `batch size = 32`
- `epochs = 50`
- 使用 `Early Stopping`

## 5. 为什么这个题目适合当前阶段

- 不要求很强的金融理论基础
- 不需要超大算力也能做出完整实验
- 容易形成清晰的表格和对比图
- 英文论文写作结构比较标准
- 适合作为学生第一篇实证型论文
